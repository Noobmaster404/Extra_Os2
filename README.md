# Extra_Os2
Данная программа предназначена для анализа дампов файловой системы FAT16

## Функционал

- Полный анализ Boot Sector
- Парсинг и отображение корневого каталога
- Поиск, чтение и анализ структуры текстового файла
- Вывод содержимого корневого каталога
- Демонстрация таблицы FAT

## Установка

Для использования программы необходимо загрузить файл zadanie2.exe и запустить его. Помимо этого, проект можно клонировать и запустить его через среду разработки pycharm или другую
